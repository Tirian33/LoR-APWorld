# LoR APWorld modified
Based on the original .apworld code by [Az-LastPenguin](https://github.com/Az-LastPenguin/LoR-APWorld).
Utilizes V0.3.1d of Az's [Library of Ruina AP Mod](https://github.com/Az-LastPenguin/LORAP/tree/main).

Implements regions to help manage blocking early. Also implements logic to prevent the game from defaulting to go mode.
